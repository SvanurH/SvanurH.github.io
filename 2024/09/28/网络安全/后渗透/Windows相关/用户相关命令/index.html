用户相关命令

用户管理:

  • net user [username]：查看用户账户信息。
  • net user [username] [password] /add：添加新用户。
  • net user [username] /delete：删除用户。
  • net user [username] /active:no：禁用用户账户。

本地组管理:

  • net localgroup：显示本地组列表。
  • net localgroup [groupname]：查看组的成员。
  • net localgroup [groupname] [username] /add：将用户添加到本地组。
    • net group administrator [username] /add添加用户到管理员组
  • net localgroup [groupname] [username] /delete：从本地组中删除用户。

权限管理:

  • icacls [文件或目录路径]：查看文件或目录的访问控制列表（ACL）。
  • icacls [文件或目录路径] /grant [用户名]:(P)(F)：授予用户对文件或目录的完全控制权限。
  • icacls [文件或目录路径] /grant [用户名]:(R)：授予用户对文件或目录的读取权限。
  • icacls [文件或目录路径] /remove [用户名]：移除用户对文件或目录的权限。

服务管理:

  • sc query：显示服务状态。
  • sc start [服务名]：启动服务。
  • sc stop [服务名]：停止服务。

进程管理:

  • tasklist：列出当前运行的进程。
  • taskkill /im [进程名]：结束指定的进程。

系统信息:

  • systeminfo：显示本地或远程主机的配置信息。

磁盘管理:

  • diskpart：启动磁盘管理脚本。

网络管理:

  • ipconfig：显示当前的TCP/IP配置。
  • ping [主机名]：测试网络连接。

事件查看器:

  • eventvwr.msc：打开事件查看器。

注册表管理:

  • regedit：打开注册表编辑器。
