#JAVA学习

Arrays

用来操作数组的一个工具类

Arrays类提供的常见方法

方法名 说明
public static String toString(类型[] arr) 返回数组的内容
public static int[] copyPfRange(类型[] arr,起始索引,结束索引) 拷贝数组（指定范围）
public static copyOf(类型[] arr,int newLength) 拷贝数组
public sattic setAll(double[] array,IntToDoubleFunction generator) 把数组中的原始数据改为新数据
public static void sort(类型[] arr) 对数组进行排序(默认是升序排序)

对数组中的数据进行排序

如果数组中存储的是对象，如何排序？

方式1

  • 让该对象的类实现Comparable（比较规则）接口，然后重写compareTo方法，自己来制定比较规则.

方式2

  • 使用下面这个sort方法，创建Comparator比较器接口的匿名内部类对象，然后自己制定比较规则。
    public static <T>void sort(T[] arr,Comparator<? super T>c)对数组进行排序（支持自定义排序规则）
