JDK8开始新增加的日期、时间

为什么要学新增加的时间。

之前的时间API
  1. 设计不合理，使用不方便，很多都被淘汰了
  2. 都是可变对象，修改后会丢失最开始的时间信息
  3. 线程不安全
  4. 只能精确到毫秒
新增加的时间API
  1. 设计更合理，功能丰富，使用更方便
  2. 都是不可变对象，修改后会返回新的时间对象，不会丢失最开始的时间
  3. 线程安全
  4. 能精确到毫秒、纳秒
