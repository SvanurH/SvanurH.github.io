#JAVA学习

面向过程更注重事情的每一个步骤以及顺序，面向对象更注重事情有哪些参与者（对象）、以及各自需要做什么。

面向过程比较直接高效，面向对象更易于复用、扩展和维护。

面向对象

封装

封装的意义，在于明确标识出允许外部使用的所有成员函数和数据项。

继承

继承基类的方法，并做出自己的改变或扩展
子类共性的方法或者属性直接使用父类的，而不需要自己再定义，只需要拓展自己个性化的

多态
