Css学习

#前端学习

Css引入的几种方式

内联

直接在标签中的style属性中写样式
<a style=""></a>

内嵌

在html文件中写入到style标签中

外联

<link rel="stylesheet" href="" />

外部引用

在一个css中引入另外一个css文件，必须在首行

@import 'test.css';

优先级

就近原则

内联>内嵌>外联>外部引用

选择器

选择器{
    属性1：值;
    属性2：值;
}

元素选择器

类选择器

id选择器

选择器优先级问题

ID选择器>类选择器>元素选择器

复合选择器

后代选择器

语法：父元素 子元素{
    }

优先级

优先级没有变化，还是ID选择器最大

子元素选择器

子元素只会选择子元素，而不会选择孙元素，但是后代选择器会选中所有后代

并集选择器

全部选中

大小单位

px

像素，是一个固定单位

em

倍数，基于浏览器默认值，16px 以倍数来计算的

百分比（%）

以浏览器默认值参照的

字体属性

margin BUG

如果对子元素使用margin，导致父元素也移位，可以在父元素上面使用
over-flow:hidden;，来避免出现。

别的

over-flow:visible;显示
transition: all 0.3s;动画过渡0.3s，过渡的效果一定是需要从开始状态向结束状态变化，必须要有开始和结束状态。否则没有过渡效果
box-shadow: x轴偏移 y轴偏移 模糊值 外延值 颜色

页面特效

transform

当translate触发动画效果移动的时候，如果它右边还有div，则会被挤开，导致两个div一起动。

用margin-left可以让其动起来，但是也可以用transform,不会挤到后续元素

animation

animation: *动画名称 *动画持续时间 动画过渡类型 延迟时间 播放次数 
            是否停留在结束状态 反向运动
动画播放次数：默认一次，infinite无限循环，过去一次回来一次
是否停留在结束状态：forwards
反向运动：alternate(播放次数必须是2次以上)

布局方式

flex
float
position
    相对定位:相对于距离自己最近的父元素进行定位
    绝对定位：定位在指定页面的位置，会随着页面动，脱离文档流
    固定定位：固定在屏幕一个位置不会变化，脱离文档流，层级更高
    当使用了定位之后，激活属性 left top right bottom | z-index (层级) -1-9999
