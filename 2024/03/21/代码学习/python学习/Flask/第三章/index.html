内容回顾

  1. 什么是接口？
    • interface类型，python没有，Java/C#才有。用于约束实现该接口的类中必须有某些指定方法。
    • api也可以成为一个接口
  2. 什么是抽象类和抽象方法？
    • 他既具有约束的功能又具有提供子类继承方法的功能。python中通过abc实现（一般不用，用异常）。
  3. 什么是重载什么是重写？
    • python中有重写没有重载
  4. flask和django的区别
  5. 什么是数据库链接池？作用是什么？
    • 内部维护几个与数据库的链接，当我们要访问数据库的时候去链接池中申请，用完再放回去
    • 如果没有，用户每次请求都要创建，用户多的话要重复创建，并且效率低
    • 如果用一个，多线程的话会有问题。
    • 使用dbutils
  6. 面向对象的 上下文管理。
