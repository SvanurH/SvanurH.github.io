MySql简介

#网络安全

  • Mysql是小型的关系型数据库，将复杂的数据以表格的形式呈现
  • 体积小
  • 速度快
  • 开源

关系型数据库有三种关系

  • 一对一
  • 一对多
  • 多对多

sql语句的四大类型

  • DDL：数据定义语句
  • DQL：数据查询语句
  • DML：数据库操作语句
  • DCL：数据库控制语句（数据库的权限设置）

创建表

数据类型

  • 数字
    • 整型int
    • 浮点类型float 4个字节
    • 双精度浮点double 8个字节
    • decimal(总位数,小数点后的位数)
  • 字符串
    • 固定长度字符串char(长度)，会删除末尾空格
    • 长度可变字符串varchar(长度)
    • 较长文本text
  • 日期
    • date yyyy-MM-dd表示年月日
    • datetime yyyy-MM-dd HH:mm:ss表示年月日时分秒

约束

  • not null 非空约束
  • default 默认值
  • primary key 主键约束
  • unique 唯一约束
  • check 检查约束check(age=='男' or age=='女')
  • foreign key 外键约束

分组查询

  • 当使用了group by的时候，select后面跟随的内容必须是聚合函数和被group by指定的内容，否则会查出无效数据

子查询

  • 相关子查询
  1. 返回只能是单行单列
  2. 子查询依赖主查询，不能独立运行
  3. 主查询执行一次，子查询也会执行一次
  • 使用from (子查询)时，因为子查询出来的表不是真正的表，所以要as

非相关子查询

  1. 先执行子查询，其结果不会被显示，而是返回给外部查询来作为外部查询的查询条件；
  2. 根据子查询的结果，执行外部查询。

数据库的三大范式

层层递进的关系

  1. 第一范式-原子性
    1. 列不可再分
    2. 行不可重复
  2. 第二范式-非主键列依赖主键列
  3. 第三范式-非主键列不依赖于其他的非主键列
