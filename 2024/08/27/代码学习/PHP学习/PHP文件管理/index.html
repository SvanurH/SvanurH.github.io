PHP文件管理

#PHP学习 #网络安全

文件包含

  • 只要有公用的文件/数据，都可以用文件包含的形式，包含进来，这样就可以减少冗余代码

文件上传

enctype="multipart/form-data" 不对字符进行编码，使用包含文件上传空间的表单时，必须使用该值

move_uploaded_file(string $from,string $to):bool

本函数检查并确保由 from 指定的文件是合法的上传文件（即通过 PHP 的 HTTP POST 上传机制所上传的）。如果文件合法，则将其移动为由 to 指定的文件。

如果目标文件已经存在，将会被覆盖。

is_dir(string filename):bool

判断给定文件名是否是一个目录

filename

文件路径，如果 filename 是相对文件名，会相对于当前工作目录进行检查。如果 filename 是符号链接或者硬链接，然后解析链接并检查。

返回值

如果文件名存在，并且是个目录，返回 true否则返回false

__DIR__

魔术方法，获取当前绝对路径

mkdir(dir,premissions,recursive,context):bool

  • 尝试新建由 directory 指定的目录。
  • premisssions默认权限是 0777,windows下被忽略
  • recursive如果是true，还会创建指定的dir的任何父级目录

文件管理

  • realpath文件目录
  • opendir打开目录
  • readdir读出目录
