PHP的OOP编程

#网络安全 #PHP学习

面向对象

  1. 封装
  2. 继承
  3. 多态
  • 属性不加修饰符一般默认是private
  • 方法不加修饰符一般是public

static

  • 静态

如果使用static修饰某个变量，那么这个变量就不属于某个对象的,而是属于这整一个类了

  • 静态方法中不能调用非静态属性变量

const

与static不同const是不可修改的
const是不能修饰函数的
const只能在类中使用，在类外定义常量要使用define,从PHP5.6开始，可以在类外使用const但是其内部还是转换为define

单例模式的实现

使用一种模式，即所有的使用都用一个对象

  • 这个类不能继承
  • 构造函数要私有化
