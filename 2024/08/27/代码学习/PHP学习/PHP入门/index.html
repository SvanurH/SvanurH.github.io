const和define的区别

  • const不能在条件语句中使用。如果要定义一个全局变量，const必须要处于整个代码的最外层
  • const只能用来声明变量（如数字、字符串，或者true, false, null, _FILE_），而define()还能接受表达式。不过在PHP5.6之后const也可以接受常量的表达式了：
  • const的常量命名只能用直白的文本，而define()允许你用任何表达式来对常量命名。这样我们就可以执行以下操作
  • const定义的常量是大小写敏感的，但是define允许你将其第三个参数设置为true来关闭其对大小写的敏感
  • const在编译时就被执行了，所以它在速度上要比define快一点。
  • 除非你需要使用表达式或者在条件语句中定义常量，不然的话仅仅是为了代码的简单可读性你都最好要使用const

字符串相关知识

在php中，

  • 字符串变量如果是双引号，则会解释其中的变量，
  • 单引号则不解释其中的变量
  • 单引号也不会解释\n转义符，双引号则会进行转义

substr(string,int start[,int length]);

返回字符串string由start和length参数指定的子字符串

字符串从0开始

strchr(string,needle);

查找指定字符在字符串中第一次出现

返回字符串中的一部分，从needle的最后出现位置开始，直到字符串末尾

needle

如果needle包含了不止一个字符，那么仅使用第一个字符。不同于strstr()
如果needle不是一个字符串，那么将被转化为整型并被视为字符顺序值
